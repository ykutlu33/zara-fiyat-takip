# Zara Fiyat Takip Botu

Bu bot Zara'daki ürün fiyatını her 5 dakikada bir kontrol eder ve fiyat değişince Telegram'dan bildirim gönderir.

## Takip Edilen Ürün
ZW Collection Hayvan Desenli Kısa Suni Kürk Kaban

## Nasıl Çalışır?
- GitHub Actions her 5 dakikada bir çalışır
- Fiyat değişirse Telegram bildirimi gelir
- Fiyat aynıysa bildirim gelmez
